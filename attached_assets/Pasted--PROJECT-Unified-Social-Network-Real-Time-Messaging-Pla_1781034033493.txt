# PROJECT: Unified Social Network + Real-Time Messaging Platform MVP

## Objective

Build a full-stack social media platform that combines the core functionality of Facebook and Telegram into a single application.

This must be a REAL working MVP using a production-grade backend, database, authentication, storage, and real-time communication.

DO NOT use dummy data, mock APIs, placeholder users, fake messages, or sample feeds.

All features must connect to real backend services and persist data in the database.

---
